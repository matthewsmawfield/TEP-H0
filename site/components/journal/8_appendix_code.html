Appendix B: Code and Data Availability

All analysis code is publicly available at github.com/matthewsmawfield/TEP-H0 (DOI: 10.5281/zenodo.18216583; licence: CC BY 4.0). The complete pipeline reproduces all results, figures, and statistics reported in this paper in under two minutes on standard hardware.

B.1 Quick Start

git clone https://github.com/matthewsmawfield/TEP-H0.git
pip install -r requirements.txt
python scripts/run_pipeline.py

The pipeline automatically retrieves all required data from public archives (SH0ES, Pantheon+, HyperLEDA, VizieR).

B.2 Pipeline Steps

The analysis is organized into ten sequential steps:

StepScriptDescription
1step_1_data_ingestion.pyDownloads SH0ES distance moduli and Pantheon+ redshifts; cross-matches with velocity dispersion catalogues
1bstep_1b_aperture_correction.pyApplies Jorgensen et al. (1995) aperture corrections to $R_{\rm eff}/8$
2step_2_stratification.pyCalculates per-host $H_0$; stratifies by median $\sigma$; computes correlation statistics
3step_3_tep_correction.pyOptimises $\alpha$ by minimising residual $H_0$–$\sigma$ slope; bootstrap uncertainty estimation
4step_4_robustness_checks.pyJackknife stability; bivariate analysis; covariance-aware significance; flow/environment controls
5step_5_m31_analysis.pyDifferential P-L analysis of M31 Cepheids (Inner vs Outer)
6step_6_multivariate_analysis.pyOLS regression controlling for age, dust, and host mass
7step_7_lmc_replication.pyControl test: LMC differential analysis (null signal expected)
8step_8_m31_phat_analysis.pyHST J/H band analysis; metallicity control via colour matching
9step_9_final_synthesis.pySynthesis figures and final summary statistics
10step_10_anchor_stratification.pyIndependent P-L fits to geometric anchors (LMC, NGC 4258, M31)

B.3 Key Output Files

After running the pipeline, the following files contain the primary results:

  • results/outputs/tep_correction_results.json — unified $H_0$, optimal $\alpha$, Planck tension
  • results/outputs/stratification_results.json — high/low-$\sigma$ stratification statistics
  • results/outputs/covariance_robustness.json — covariance-aware $p$-values and $N_{\rm eff}$
  • results/outputs/out_of_sample_validation.json — train/test and LOOCV results
  • data/processed/hosts_processed.csv — complete host galaxy catalogue with $\sigma$, $H_0$, and corrections

B.4 Dependencies

The pipeline requires Python 3.8+ with the following packages: numpy, scipy, pandas, matplotlib, astropy, astroquery.
