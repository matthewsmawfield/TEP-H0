# Processed Data Files

These files are the final pipeline outputs used for statistical analysis.

## Files

- `hosts_processed.csv` — Final processed host data with all derived quantities
- `hosts_metadata_enriched.csv` — Enriched metadata for all hosts

## Data Sources

- **Distances / redshifts:** `data/raw/Pantheon+SH0ES.dat` (Scolnic+2022)
- **Velocity dispersions:** `data/raw/external/velocity_dispersions_literature.csv` (master file with ADS bibcodes)
- **Coordinates / properties:** Generated from literature names and VizieR queries

## Important

These files are regenerated by `scripts/run_pipeline.py`. 
The authoritative inputs are in `data/raw/` and `data/raw/external/`.
