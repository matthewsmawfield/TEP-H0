# Interim Data Files

These files are generated by the pipeline during processing.
They are intermediate outputs, not authoritative inputs.

## Files

- `hosts_coords.csv` — Host galaxy coordinates (RA/Dec) from name resolution
- `hosts_properties.csv` — Host galaxy properties from metadata compilation
- `hosts_kinematics_hyperleda.csv` — HyperLEDA kinematic query results
- `kodric_hst_cepheids.csv` — M31 Cepheid data from Kodric et al. (2018)
- `r22_distances.csv` — Distance moduli from Riess et al. (2022) SH0ES
- `reconstructed_shoes_cepheids.csv` — Reconstructed SH0ES Cepheid sample

## Important

These files are regenerated by the pipeline. Do not treat them as primary data sources.
The authoritative inputs are in `data/raw/` and `data/raw/external/`.
